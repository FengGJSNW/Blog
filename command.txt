=============================================
# 查看当前分支和状态
git status

# 查看远程仓库配置（如果没有则显示为空）
git remote -v

# 添加远程仓库（命名为 origin）
git remote add origin https://github.com/FengGJSNW/Blog.git

# 如果已存在 origin，先删除再添加
git remote remove origin
git remote add origin https://github.com/FengGJSNW/Blog.git

# 初始化仓库（如果尚未初始化）
git init

# 添加所有文件到暂存区
git add .

# 或选择性添加特定文件
# git add README.md index.html css/

# 提交更改
git commit -m "Initial commit: Blog project"

# 创建并切换到 main 分支（GitHub 默认分支）
git branch -M main

# 推送到远程仓库
git push -u origin main
